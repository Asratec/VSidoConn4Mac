# VSidoConn4Mac
V-Sido CONNECT Sample Code For Mac OS X  


ビルド済みバイナリをインストールする。  
  https://v-sido-developer.com/files/pdf/150514001006001.pdf  

	
ソースコードをビルドする.  
1.「Xcode Command Line Tools」ををインストールする。  
2.「CMake」ををインストールする。  
3.ソースコードをチェックアウトする  
  git clone https://github.com/Asratec/VVSidoConn4Mac.git  
4.依存ソースコードをチェックアウトする  
  cd VSidoConn4Mac  
  git clone https://github.com/Asratec/VSidoConnServer.git  
5.パッケージをビルドする。  
  make package    
  成果物はVSidoConn4Mac.tar.gzとなる。
